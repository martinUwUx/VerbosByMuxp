# **App Name**: Verbs By Muxp

## Core Features:

- Custom Verb Selection: Users can select verbs for review by category (regular, irregular, combined), then select individual verbs or choose a 'random 20 verbs' option.
- Configurable Quiz Format: Users choose which verb forms (Infinitive, Past Simple, Past Participle) to include in the quiz.
- Interactive Quiz Interface: Presents verbs in a table format with Spanish translations as hints, and empty fields for users to input the correct verb forms.
- Real-time Scoring: Automatically check the provided values against a datastore. Each correct verb form is worth one point, and the application calculates a total score and percentage.
- Feedback Display: Clear presentation of quiz results with total score, percentage of correct answers, and visual feedback (correct/incorrect) for each verb form.
- Answer Key: Display the correct answer next to incorrect user submissions for immediate learning reinforcement.
